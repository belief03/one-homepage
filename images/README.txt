・hero-bg.jpg … ヒーローセクションの背景画像（横長推奨）
・character.png … ヒーロー用イメージキャラクター（正方形推奨）
・philosophy.jpg … 想いセクションの代表写真（正方形推奨）
・flow1.png ～ flow4.png … ご依頼の流れ（①ご相談 ②お見積り ③ご契約 ④納品・サポート）の写真（正方形推奨・140×140px程度） … 依頼の流れ（①ご相談 ②お見積り ③ご契約 ④納品・サポート）の写真（正方形推奨・140×140px程度）

画像がなくてもサイトは表示されます（画像部分だけ枠で表示されます）。

--- 想い用写真（philosophy.jpg）の入れ方 ---
1. チャットで送った「誠に生きる」の写真を、右クリック →「名前を付けて画像を保存」
2. 保存したファイル名を philosophy.jpg に変更
3. この images フォルダに philosophy.jpg を入れる
4. ブラウザで index.html を再読み込みして確認
