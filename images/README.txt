このフォルダに次の画像を置くと、サイトに表示されます。

・hero-bg.jpg … ヒーローセクションの背景画像（横長推奨）
・hero-bg-2.png … ヒーロー用の2枚目背景（フェードで切り替え。横長推奨）
・character.png … ヒーロー用イメージキャラクター（正方形推奨）
・philosophy.jpg … 想いセクションの代表写真（正方形推奨）
・flow1.png ～ flow4.png … ご依頼の流れの写真（正方形推奨・140×140px程度）
・logo-one.png … フッター用「One」ロゴ（横長・高さ40px前後推奨）
・footer-bg.png … フッター背景（Oneが中央に来るロゴ画像。横長推奨）

画像がなくてもサイトは表示されます（画像部分だけ枠で表示されます）。

--- フッター用 One ロゴ（logo-one.png）の作り方 ---
1. ヘッダー用画像（hero-bg-2.png）を画像ソフトで開く
2. 左上の「One」（犬のO＋ne＋矢印）と「System Engineer」の部分を選択して切り抜く
3. 背景を透過するか、余白を少なめにして保存
4. ファイル名を logo-one.png にし、この images フォルダに入れる
5. ブラウザで再読み込みしてフッターのロゴ表示を確認
（画像がない間はフッターにロゴ画像の枠だけ表示されます）

--- 想い用写真（philosophy.jpg）の入れ方 ---
1. チャットで送った「誠に生きる」の写真を、右クリック →「名前を付けて画像を保存」
2. 保存したファイル名を philosophy.jpg に変更
3. この images フォルダに philosophy.jpg を入れる
4. ブラウザで index.html を再読み込みして確認
